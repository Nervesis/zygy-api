# Gradio App

This is a simple Gradio application built in `app.py`.

## Requirements

Make sure you have:

-   Python 3.8 or higher installed
-   `pip` available on your system

All required Python packages are listed in `requirements.txt`.

Install dependencies:

    ``` bash
    pip install -r requirements.txt
    ```

------------------------------------------------------------------------

## Running the App

You can run the app in two different ways:

### Option 1: Run with Python

``` bash
python app.py
```

### Option 2: Run with Gradio CLI

``` bash
gradio app.py
```

------------------------------------------------------------------------

## Accessing the App

After running the command, you should see output in the terminal showing
a local URL, usually something like:

    Running on local URL: http://127.0.0.1:7860

Open that URL in your browser to use the app.
