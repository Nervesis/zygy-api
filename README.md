# Zygy API -- How to Obtain Your API Key

This guide walks you through the process of generating an API key for
the Zygy API service.

------------------------------------------------------------------------

## 🚀 Step-by-Step Instructions

### 1️⃣ Access the Zygy Business Portal

Go to:

    https://biz.zygy.com

Log in to your existing account.\
If you do not have an account yet, complete the registration process
first.

------------------------------------------------------------------------

### 2️⃣ Open the Admin Panel

1.  After logging in, locate your **profile icon** in the top-right
    corner.
2.  Click on the profile icon.
3.  Select the **Admin** tab from the dropdown menu.

------------------------------------------------------------------------

### 3️⃣ Navigate to API Key Management

1.  Inside the Admin section, click on **Manage API Key**.
2.  Click the **Create New Secret Key** button.

------------------------------------------------------------------------

### 4️⃣ Name Your Secret Key

-   Provide a clear and descriptive name for your secret key.\
    (Example: `Production-Backend`, `Testing-Environment`, etc.)

------------------------------------------------------------------------

### 5️⃣ Save Your API Key Securely ⚠️

After generating the key:

-   **Copy the secret key immediately.**
-   Store it in a secure location such as:
    -   A password manager
    -   A secure vault
    -   An encrypted configuration file

> ⚠️ Important: The secret key will NOT be displayed again after this
> step.

If you lose the key, you will need to generate a new one.

------------------------------------------------------------------------

## 🔐 Security Best Practices

-   Never expose your API key in public repositories.
-   Do not embed keys directly in frontend applications.
-   Rotate your keys periodically.
-   Revoke unused or compromised keys immediately.

------------------------------------------------------------------------

## 📞 Need Help?

If you encounter issues generating your API key, contact Zygy support
through the business portal.

------------------------------------------------------------------------

**You're now ready to integrate with the Zygy API! 🎉**
